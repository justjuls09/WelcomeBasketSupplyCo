1. Human Output (What the Barista Sees)
The human-facing output of this system is a clear, consistent, and easy-to-read cup label used by the barista or delivery coordinator.
Human Outputs Include:
- Drink Name (Housewarming Latte, Welcome‑Home Americano, Cozy Condo Cappuccino, Townhouse Toasted Mocha, New‑Keys Nitro Brew)
- Size (Small, Medium, Large)
- Milk Type (Dairy, Oat, Almond, Soy, Coconut)
- Syrup Pumps (0.10 each)
- Safety Notes (allergen confirmation)
- Delivery Location (office number, department, or home unit)
- Total Price
- Order ID / Timestamp

3. The Problem (My “Soy Milk Incident”)
During a new-home welcome event, a mislabeled drink containing soy milk was delivered to a customer with a severe soy allergy. This incident highlighted the need for a strict allergen-safe ordering system with clear milk tracking, accurate labeling, and reliable delivery information.
4. The Resilience (Most Important Technical Sprint)
The most important technical sprint is the Files Sprint. The system must reliably store drink orders, milk selections, syrup counts, and delivery locations. File handling ensures consistency, prevents allergen errors, and allows the system to “remember” customers.
